## Cómo añadir entradas

Cuando en FASE 1 se encuentre un ID de Pexels válido para un lugar turístico conocido (capitales, lagos famosos, monumentos icónicos), añadirlo aquí además de en `datos-enriquecidos/logistica.md`. Así estará disponible para futuros viajes que pasen por el mismo lugar.

Formato para añadir:
```
| [Nombre lugar] | [ID] | [descripción breve de la foto] | ✅ |
```
